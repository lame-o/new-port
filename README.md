# Portfolio Website

A modern, interactive portfolio website built with React, Three.js, and Vite. Features a unique dithered wave background effect and a dynamic dock-style navigation.

## Features

- Interactive 3D background with dithered wave effect
- macOS-style dock navigation
- Responsive design
- Glass-morphism UI elements

## Tech Stack

- React
- TypeScript
- Three.js
- Framer Motion
- Vite

## Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Building

To create a production build:

```bash
npm run build
```
